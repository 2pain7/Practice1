# Practice1
Система управления магазином обуви, разработанная на Avalonia UI с использованием PostgreSQL и Entity Framework Core.

Десктопное приложение для автоматизации работы обувного магазина. Позволяет управлять каталогом товаров, заказами, пользователями и поставщиками.

Технологии:
Avalonia UI 11.0.10 - кроссплатформенный UI фреймворк
PostgreSQL - база данных
Entity Framework Core 8.0.4 - ORM для работы с БД
Npgsql - провайдер PostgreSQL для EF Core
.NET 8.0 - платформа разработки

Функциональность:
Авторизация пользователей с разными ролями (Администратор, Менеджер, Авторизированный клиент)
Управление товарами - просмотр, поиск, фильтрация
Категории товаров (Мужская/Женская обувь)
Поставщики и производители - управление справочниками
Сортировка товаров по алфавиту
Поиск товаров по названию
Отображение изображений товаров
Управление заказами и статусами (Новый, Готов, Завершен)

Структура базы данных:
-- Основные таблицы
- users - пользователи системы
- products - товары
- suppliers - поставщики
- manufactures - производители
- orders - заказы
- order_details - детали заказов
- pick_points - пункты выдачи


Установка и запуск
Требования:
.NET 8.0 SDK
PostgreSQL 12 или выше
